Note on naming

The ZombieVerter uses "pot" for both types — it's a historical naming holdover. Calibration procedure is identical regardless of sensor type.

C04 — CALIBRATION

Calibration Procedure7 steps with the web interface open

1
Open Spot Values → enable Auto Refresh
Watch pot and pot2 while moving pedal — both must change
2
Pedal fully released — note resting ADC
e.g. pot = 435 at rest
3
Pedal fully pressed — note WOT ADC
e.g. pot = 2410 at WOT
4
Set potmin = resting + 10
435 + 10 = 445
5
Set potmax = WOT − 10
2410 − 10 = 2400
6
Repeat for pot2min / pot2max (dual channel)
7
Set throtmax = 100, throtmin = 0
throtmax DEFAULTS to 0 — this step is mandatory

Verify with potnom

Pedal positionpotnom should show
Fully released0 (or very close)
Full travel100 (or very close)
MovementSmooth, proportional
⚠ potnom non-zero at rest?

potmin set too low. Increase potmin by 5–10 counts until potnom reads 0 at rest.

⚠ potnom never reaches 100?

potmax set too high. Decrease potmax by 5–10 counts.

Dual channel plausibility

With potmode=dual, both channels are cross-checked continuously. Both must be calibrated or plausibility faults trigger immediately.

C04 — PARAMETERS

throtmax & Throttle ParametersThe parameter everyone forgets — and the ones that tune it

0
throtmax default value
= 0% maximum torque output
= throttle does absolutely nothing
🔴 Set throtmax = 100

This is the single most common cause of "throttle does nothing after calibration." The default is intentionally zero — it requires a conscious decision from the builder. Set it to 100 before testing.

Full Parameter Reference

ParameterDescription
throtminMinimum potnom. Set negative for throttle-lift regen (e.g. -5 = light regen when foot off)
