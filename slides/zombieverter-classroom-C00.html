C00 — THE BIG BREAK

V2.00A — The Major BreakMay 2023 — treat all V1 content with caution

What Changed in V2.00A

IO Matrix
Configurable input/output pin assignments via web UI. Before V2, many pins had fixed functions. Din/Dout sections in web UI are V2 only.
Module Decoupling
Leaf stack components (inverter, PDM, charger) can now be used independently. V1 required all together.
Parameter Reorganisation
Many parameters renamed, relocated, or removed. V1 forum posts with specific parameter names may be wrong for V2.

New in V2.20A (Dec 2024)

🔴 Upgrade warning — ALL major versions

After any major firmware upgrade, you MUST:
1. Reselect inverter type in Parameters → Comms
2. Save to flash
3. Reboot VCU
Toyota inverter users must reboot after upgrade or inverter won't initialise.
