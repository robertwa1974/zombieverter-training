Permanent PowerPin 56 and Pin 55 — always on

Fuse Block Strategy

Fuse blockPowers
Permanent blockVCU Pin 56, ISA shunt — direct from battery
Ignition ON blockDevices that power on with key — fed from Pin 32 relay
Start position blockStart signal only — momentary feed
Bench supply

Set bench PSU to 13V, limit to 1–2A. A 9V battery works for basic power-up verification only — not functional testing.

Why Permanent Power?

Charge timer management
VCU manages scheduled charging when parked
ISA shunt continuity
Shunt needs same permanent feed — must stay powered to maintain UDC reporting
150mA idle — negligible drain
Won't flatten a healthy 12V battery during normal parking
🔴 Never ignition-switch Pin 56

The VCU loses UDC history when key is off. Stale or zero UDC causes precharge failures on next startup.

W06 — IGNITION & START

Ignition ON & StartTwo separate signals — sustained vs momentary

Ignition ON — Pin 15

PinSignal typeWhen
Pin 15Sustained 12VKey in ON position — stays high while driving

Tells VCU the vehicle is "on" but does NOT start contactor sequencing by itself.

Bench test

Toggle switch between 12V+ and Pin 15. Toggle on = key to ON position.

Start — Pin 52

PinSignal typeDuration
Pin 52Momentary 12V pulse≥500ms — like turning a key to START
⚠ Momentary only

The start signal must be momentary — NOT sustained. Wiring it as a sustained signal can cause issues re-triggering precharge after a fault.

The Inverter Enable Relay (Pin 32)

VCU controls a relay via Pin 32 to provide managed "ignition" to the inverter, charger, and DC-DC. These devices should NOT be powered until the VCU is ready.

# Inverter enable relay wiring 12V+ ─── [Relay coil +] [Relay coil −] ─── Pin 32 (VCU) 12V+ ─── [Relay switch COM] [Relay switch NO] ─── Inverter enable └── Charger enable └── DC-DC enable
The "Holy Trinity" for run mode

Pin 56 (permanent 12V) + Pin 55 (ground) + Pin 15 (ignition ON) + Pin 52 (start pulse) + Pin 54 or 53 (direction) = minimum for run mode

Torque direction

Forward = positive torque command. Reverse = negative torque command. No physical reverse gear needed in most EV transaxles (Leaf, Outlander, Prius). GS450h is an exception — see I03.

Brake Signal

PinSignal
Pin 4912V when brake pedal pressed
🔴 Safety interlock

din_brake must be "off" for any torque output. Wire to normally-open (NO) terminal of brake light switch.

⚠ Floating input

If Pin 49 is unwired, it may pick up electrical noise and read "on" intermittently, blocking torque. Add a 10kΩ pull-down from Pin 49 to ground if you have noise issues.
