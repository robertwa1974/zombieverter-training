3
Connect to Wi-Fi AP on your device
SSID: "inverter" (or "zom_vcu") · Password: inverter123
4
Open browser → 192.168.4.1

⚠ If browser can't reach 192.168.4.1

Some recent boards don't auto-assign IP via DHCP. Manually set your device: IP = 192.168.4.2, subnet = 255.255.255.0, gateway = 192.168.4.1

C02 — THE CORE CONCEPT

Parameters vs Spot ValuesThe most important distinction in the entire interface

Parameters

What you SET. Your inputs to the system. Configurable settings that tell the VCU how to behave.

inverter type · udcsw · potmin/potmax · shuntType · throtmax · CAN assignments
🔴 Save to Flash

Changes in the Parameters tab are NOT saved until you use Commands → Save Parameters to Flash. Power-cycle without saving = all changes lost.

Spot Values

What the system REPORTS back to you. Real-time read-only outputs.

opmode · udc · pot/pot2 · potnom · din_brake · din_forward · invstat · status
Auto Refresh

Enable Auto Refresh in Spot Values for live monitoring. For calibration, use manual Refresh to freeze values while reading them.

C02 — SPOT VALUES

Key Spot ValuesWhat to watch and when

For Precharge Diagnosis

Spot valueWhat it shows
opmodeoff / precharge / run / charge
udcHV bus voltage from shunt — must be non-zero
statusLast status message — WHY something happened
invstatInverter status — "on" in run mode

For Throttle Calibration

Spot valueWhat it shows
potRaw ADC channel 1 — watch while moving pedal
pot2Raw ADC channel 2
potnomNormalised 0–100 — should be 0 at rest, 100 at WOT
torqueRequested torque — confirms throttle map is working

For Direction/Safety

Spot valueWhat it shows
din_brakeBrake signal — must be "off" for torque
din_forwardForward selected — must be "on" for forward torque
din_reverseReverse selected
din_startStart signal — momentary

Tips

Ctrl+F

The web interface is a single long page. Use browser find (Ctrl+F) to jump directly to any parameter. Search "udcsw" to jump straight to it.

Visible filter

Check "Visible" only for the spot values you're watching — hides clutter during diagnosis and calibration.
