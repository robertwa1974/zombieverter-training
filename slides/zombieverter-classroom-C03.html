C03 — STEPS 3 & 4

CAN Assignment & Voltage Limits

Step 3 — CAN Bus

The VCU has three CAN buses. Assign each device to the correct bus.

inverterCan = CAN1 # or wherever inverter wired shuntCan = CAN1 # same bus as shunt
⚠ Known Issue

CAN1 and CAN2 labels are reversed in some older wiring diagrams. If inverter won't communicate, try swapping the bus assignment.

Step 4 — Voltage limits

ParameterBenchReal vehicle
udcmin20V~80% of min pack V
udclim500VJust above max pack V
udcsw15V~95% of nominal pack V
🚫 Most common bench failure

udcsw defaults to 330V. With no HV connected you'll never reach it. Set it to 15V for bench testing or precharge will never complete.

C03 — STEP 5

Throttle Setup The most missed parameter in all of ZV

Calibration Method

1
Watch "pot" in Spot Values
Enable Auto Refresh. Slowly move throttle pedal through full travel.
2
Note resting value
E.g. ADC reads 200 at rest
Set potmin = 250 (slightly above)
3
Note full-travel value
E.g. ADC reads 3800 at WOT
Set potmax = 3750 (slightly below)
ZV Specific

Unlike most inverters, set pot range INSIDE mechanical travel. This ensures "valid range" is only true when pedal is genuinely moving.

throtmax — Do Not Miss This

0
throtmax default value
= 0% max torque output
= throttle does nothing
🔴 The single most common first-start failure

throtmax defaults to 0. This is intentional but always forgotten. Set it to 100 before testing.

throtmin = 0 # or negative for lift-regen throtmax = 100 # ← SET THIS. Default is 0!
C03 — STEPS 6 & 7

Brake & Direction Verify before attempting precharge

Step 6 — Brake Signal

Check din_brake in Spot Values.

Brake statedin_brake showsTorque allowed?
Not pressedoff✓ Yes
Pressedon✗ No
⚠ If stuck "on" with no brake pressed

Check if brake switch is N/C (inverts logic). Check for floating input — add pull-down to ground. Check wiring on Pin 49.

Step 7 — Direction Inputs

A direction MUST be selected for torque output.

# Apply 12V → check spot value Pin 54 → din_forward = "on" Pin 53 → din_reverse = "on"
🔴 Common cause of "torque won't work"

If neither din_forward nor din_reverse is "on", the VCU will not produce torque even in run mode.

💡 Tip

For bench testing, just hold a wire to 12V on Pin 54 while monitoring spot values.

C03 — STEPS 8 & 9

Precharge Test The moment of truth

Procedure

1
Apply Ignition ON
Pin 15 → 12V. opmode stays "off".
2
Brief START pulse
Pin 52 → momentary 12V. At least 500ms. Like turning a key — don't hold it.
3
Watch opmode in Spot Values
off → precharge → run
4
Verify throttle
Apply throttle. Watch torque spot value increase. No motor movement expected.

If it doesn't work

Stays in "precharge"

udcsw is higher than UDC reading. Lower udcsw to 15V for bench.

"precharge failed"

UDC reading is 0. Shunt is not communicating — check shuntType, shuntCan, and wiring.

Never reaches "run"

Check: din_brake = off? din_forward or din_reverse = on? UDC above udcmin?

Throttle does nothing in run

throtmax is probably still 0. Set to 100 and save to flash.
